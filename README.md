## :+1: 프로젝트 소개
__Moi__ 는 어렵기만 한 __이력서 작성__ 의 고민을 하는 취준생들을 위해 고안된 서비스입니다.<br/><br/>
__Moi__ 와 함께라면 누구나 간단하고 빠르게 전문적인 __이력서__ 를 작성 할 수 있습니다.<br/><br/>
흥부에게 박씨를 물어다준 제비처럼 __Moi__ 가 당신에게 합격을 물어다 드리겠습니다.
